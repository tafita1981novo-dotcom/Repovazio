📋 Como pegar suas credenciais (passo a passo)
Mostrar instruções YouTube OAuth
  1. Crie ou acesse o canal no YouTube com o email novo
  2. Vá em console.cloud.google.com → New Project → "psicologia-doc"
  3. APIs & Services → Library → ative YouTube Data API v3
  4. OAuth consent screen → External → preencha nome e email
  5. Credentials → Create Credentials → OAuth 2.0 Client IDs → Web application
  6. Authorized redirect URI: https://developers.google.com/oauthplayground
  7. Copie Client ID e Client Secret que aparecem
  8. Vá em developers.google.com/oauthplayground → engrenagem ⚙ → "Use your own OAuth credentials" → cole Client ID/Secret
  9. Selecione scopes: https://www.googleapis.com/auth/youtube.upload + https://www.googleapis.com/auth/youtube
  10. Authorize APIs → faça login com email novo → autorize
  11. Exchange auth code for tokens → copie o Refresh Token
  12. Channel ID: youtube.com/account_advanced (formato UCxxxx)
